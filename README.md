# COVID detection by audio and visual modalities using MAFnet


## Requirements
* Python-3.6
* Tensorflow-gpu-1.15
* Keras
* Scikit-learn

* pillow
* resampy
* ffmpeg
* pickle

## Training and testing scripts

To train the network:
```
python train.py --train
```

To test the network:
```
python train.py
```
